## Built With

## Installation

1. Clone this Repository

```sh
  https://github.com/NawiOne/booking_app.git
```

2. Install Package

```sh
  npm install
```

```sh
  npm install -g nodemon 
```


## Usage

1. Use nodemon to running the server (recommended)

```sh
  nodemon
```


2. Endpoint

```sh
  POST --> /book
  GET --> /book?booking_no&booking_date
```
